# SigilAGI Video Recording Checklist

## Target Length

3 minutes or less.

## Recording Order

### 1. Title Screen

Say:

SigilAGI is object-to-glyph detection for symbolic AI reasoning.

### 2. Problem

Say:

Most vision systems stop at captions. SigilAGI creates a symbolic reasoning trace that humans can inspect.

### 3. Core Transformation

Show:

object → glyph → meaning → relationship → reasoning trace

### 4. Demo Input

Open the live demo or local demo.

Enter:

person, phone, door, wallet, paper

### 5. Demo Output

Show:

☉ → ☏ → ⛩ → ▨ → ▧

### 6. JSON Report

Show that each object has:

- label
- glyph
- meaning
- role
- risk
- relationship trace

### 7. Reasoning Layer

Show the Gemma-ready reasoning output:

- Detected Scene
- Glyph Interpretation
- Possible Meaning
- Unknowns
- Human Review Notes

### 8. Safety Boundary

Say:

SigilAGI does not prove identity, intent, guilt, legal status, medical status, or confirmed danger. It creates a structured symbolic trace for human review.

### 9. Closing

Say:

SigilAGI turns object detection into symbolic reasoning that humans can inspect, verify, and improve.
